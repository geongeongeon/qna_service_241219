속성 설명 값
paging.isEmpty 페이지 존재 여부 (게시물이 있으면 false, 없으면 true) [[${paging.isEmpty}]]
paging.totalElements 전체 게시물 개수
paging.totalPages 전체 페이지 개수
paging.size 페이지당 보여 줄 게시물 개수
paging.number 현재 페이지 번호 (0부터 시작)
paging.hasPrevious 이전 페이지의 존재 여부 (존재하면 true, 존재하지 않으면 false)
paging.hasNext 다음 페이지의 존재 여부 (존재하면 true, 존재하지 않으면 false)
